Meta Transparency, 2019-2023

https://transparency.fb.com/metasecurity/threat-reporting

La source empirique la plus dense sur les méthodes de détection opérationnelles. Chaque takedown documente les vecteurs utilisés : clustering de graphe, analyse de micro-timing, scoring comportemental. Les opérations détectées l'ont été par les statistiques de second ordre identifiées dans le segment 05, densité de liens artificielle, synchronisation trop étroite. Les opérations non-détectées jusqu'à un signal externe confirment le raisonnement central du segment 07 : la patience reste le seul vecteur symétrique à cette architecture.
